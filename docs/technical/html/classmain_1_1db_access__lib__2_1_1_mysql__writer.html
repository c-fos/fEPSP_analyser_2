def dbWriteSignalProperties(self, ptp_, snr_, std_, mainLevel): newSigProp = db.SignalProp(snr_, std_, ptp_, mainLevel, self.idRecord ) self.session.add(newSigProp) self.session.commit() Сохранение уровня возникшей в процессе выполнения анализа ошибки.

Аргументы
soft1 - была не критическая ошибка, иначе 0
hard1 - была критическая ошибка, иначе 0
