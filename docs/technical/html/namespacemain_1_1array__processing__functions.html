Возвращает локальное значение ptp() в массиве

Сначала немного сглаживает массив скользяшим средним для того чтобы исключить случайные выбросы. Затем проходит по массиву указанной рамкой и расчитывает ptp(). Возвращает максимальное из расчитанных значений.

Аргументы
dataВходящий массив
framesizeразмер рамки считывания

def main.array_processing_functions.stdArray (   sample,
  frame 
)

Формирует массив значений стандартного отклонения

Проходит рамкой размера 2*frame по переданному в функцию массиву и расчитывает стандартное отклонение, которое записывает в результирующий массив. Размеры входящего и исходящего массивов одинаковы

Аргументы
sampleвходящий массив
frameполовина размера рамки считывания
def main.array_processing_functions.stdFinder (   data,
  frameSize,
  mean = False,
  maxStd = False 
)

Возвращает минимальное и максимальное локальные стандартные отклонения в массиве

Проходит по переданному массиву рамкой считывания и расчитывает std() и mean(). В зависимости от переданных агрументов может вернуть минимальное стандартное отклонение, максимальное стандартное отклонение, минимальное стандартное отклонение и среднее той рамки где std было минимальным.
