Загрузка массива из файла

Обрабатываются два типа файлов: записаные в виде столбца чисел в текстовом файле и сохраненный в бинарном виде массив типа int16. Тип определяется по mime типу файла. Если он текстовый, то читается как текст, в противном случае читается как бинарный.
